## Install
1. Add this two lines to the composer.json
```
"repositories": [
  {
      "type": "vcs",
      "url": "https://github.com/Relluem94s/RelluDatabase.git"
  }
]
```

```
"require": {
    "relluem94/relludatabase": "master"
}
``` 
2. Run ```composer update``` to download the RelluDatabase Dependency for your project.
